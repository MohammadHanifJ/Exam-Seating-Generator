# Admin Workflow

1) Upload Students
- Upload CSV/XLSX
- Duplicate roll numbers are ignored

2) Review & Approve
- Approve or block students
- Only approved students are used in seating generation

3) Generate Seating
- Select exam type
- Add explicit Branch + Year groups
- Select rooms and capacities
- Assign invigilators (optional)

4) Preview
- Visual grid + table
- Export combined or per-room PDF
- Send PDF by email
